<h1>Introduction</h1>
<div class="label2">Overview</div>
<br />
<p>
  LiteSpeed Web ADC (LSLB) is an affordable, high-performance layer 7 HTTP smart load-balancer application.
  LiteSpeed Web ADC is feature-rich, secure, and efficient, offering more flexibility than similarly-priced load
  balancing software. This makes it an excellent choice for small enterprises looking to scale their applications beyond
  one server â€” both to improve service speed as well as reliability in case of hardware failure.
</p>

<div class="label2">Features</div>
<ul>
  <li>Supports QUIC, HTTP/2, TLSv1.3, and TCP_FASTOPEN</li>
  <li>HTTP/1.1, HTTP/1.0 backward compatible</li>
  <li>Supports HTTP, LiteSpeed SAPI, FastCGI and AJPv13 back ends</li>
  <li>Load balance algorithms: round-robin, least load, least session</li>
  <li>High Availability IP failover</li>
  <li>Cross-data-center replication</li>
  <li>Directing request based on domain names, request URL, Cookie, SSL Session, etc.</li>
  <li>Content aware: route request based on request content</li>
  <li>Dynamic response compression/decompression (gzip)</li>
  <li>Gzip compression with backend HTTP Server</li>
  <li>Automatic HTTP protocol upgrade/downgrade to maintain persistent connection with backend servers</li>
  <li>Massive shared hosting: load balance to millions of websites</li>
  <li>Web app acceleration via LSCache with ESI</li>
  <li>URL rewrite</li>
  <li>SSL acceleration</li>
  <li>Geotargeting support</li>
  <li>IPv6 support</li>
  <li>Layer 7 anti-DDoS filtering</li>
  <li>Web Application Firewall (HTTP firewall), filter attacking requests based on request content</li>
  <li>Chroot for enhanced security</li>
  <li>Backend server health monitoring</li>
  <li>Web administration console</li>
  <li>Online upgrade to keep your server up-to-date</li>
  <li>Simplified deployment via ZeroConf</li>
</ul>
